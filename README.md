# Task: Top Cuisines Analysis

# Objective:
Identify the top three most common cuisines in the dataset and calculate
the percentage of restaurants serving each of these cuisines.

# Approach
- Removed missing cuisine values
- Split multiple cuisines listed for a restaurant
- Counted individual cuisine frequencies
- Calculated percentage contribution of each top cuisine
  
# Conclusion
North Indian, Chinese, and Fast Food cuisines dominate the dataset,
indicating high customer demand for these cuisine types.
